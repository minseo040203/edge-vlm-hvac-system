# Pivot or Persevere Decision

## 결정

**Decision: Persevere**

HVAC Dashboard V2 실험 결과, 핵심 지표가 기존 Dashboard 대비 개선되었으므로 현재 방향을 유지한다.

---

## 판단 근거

| 지표 | 결과 |
|------|------|
| Task Completion Rate | +17.1% |
| CTA Click Rate | +54.2% |
| Feedback Rating | +23.5% |
| 사용자 평균 만족도 | 4.0 / 5.0 |

---

## 사용자 피드백 근거

### 긍정 피드백

- 핵심 지표가 한눈에 보여 빠르게 상태를 판단할 수 있었다.
- 에너지 절감 추천이 의사결정에 도움이 되었다.
- 추천 조치가 명확해서 현장 대응에 도움이 되었다.

### 부정 피드백

- 전문 용어 설명이 부족하다.
- 모바일 화면에서 일부 정보가 많아 보인다.
- 성과 요약과 데이터 내보내기 기능이 필요하다.

---

## 결론

Dashboard V2는 사용자 작업 완료율과 상호작용 지표를 개선하였다.  
따라서 제품 방향은 **Persevere**로 결정한다.

단, 다음 개선 과제를 후속 백로그로 등록한다.

1. Tooltip 기반 용어 설명
2. 모바일 compact layout
3. KPI 주간 요약 카드
4. CSV export 기능
5. 위험 구역 자동 정렬
