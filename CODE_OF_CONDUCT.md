# Code of Conduct

## 우리의 약속

본 프로젝트는 누구나 안전하고 존중받는 환경에서 참여할 수 있도록 한다.

참여자는 다음 원칙을 지킨다.

1. 서로를 존중한다.
2. 공격적이거나 차별적인 표현을 사용하지 않는다.
3. 건설적인 피드백을 제공한다.
4. 협업 과정에서 투명하게 소통한다.
5. 프로젝트의 목적과 학습 목표를 존중한다.

---

## 허용되지 않는 행동

다음 행동은 허용되지 않는다.

- 모욕, 비하, 혐오 표현
- 괴롭힘 또는 위협
- 개인정보 무단 공개
- 악의적 코드 삽입
- 타인의 기여를 부당하게 삭제하거나 왜곡하는 행위

---

## 신고 및 조치

문제가 발생하면 저장소 Issue 또는 관리자에게 보고한다.

관리자는 다음 조치를 취할 수 있다.

1. 경고
2. 댓글 또는 PR 제한
3. 기여 제한
4. 저장소 접근 제한

---

## 적용 범위

본 행동 강령은 GitHub Issues, Pull Requests, Discussions, 문서, 코드 리뷰 등 프로젝트와 관련된 모든 공간에 적용된다.
